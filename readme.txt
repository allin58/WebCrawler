This project is implemented for grabbing and analysing data which is got from web site.
 For using you must specify web pages which will be parsed.
You can do it in the data/in/pages.txt For example:
https://www.google.com/
https://coinmarketcap.com/

Also you must set terms in the data/in/terms.txt For example:
Market
coin

Now you can start application, for that execute build.bat
You will find the results in the folder  data/out

You can change all input and output paths in application.properties

Application operation is logged to log file.
